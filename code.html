Spotify Decades



I wanted to see the decade distribution of my Spotify library and was disappointed to see that existing Spotify stats websites don't give that information. So naturally I made my own tool.


See the code on Github

How it works

This application was made using Spring Boot. It communicates with the Spotify API using HTTP. The user signs in with their Spotify account which causes an authorization code to be obtained and then exchanged for an access token which will be used for all user data requests. The app fetches JSONObjects of the user's liked songs from which the release year of each song is obtained. A new Decade object is made or a tally is added to an existing one. The decade data is stored in an arraylist. The user can view their results as a list or graphed.



Turnfable



This is a medieval-themed multiplayer party game in which players assume the role of scribes and take turns assembling a story out of randomized words. Created as a group project for IMD2006: Intro to Game Design and Development. I did most of the programming work for the game.


See the code on Github Play it on Itch.io

How it works

Players come up with words of different types to be added to a communal word pool. From this pool each player receives a selection of words that they use to fill in slots from pre-built sentence templates. Partway through the game, each player is given the chance to swap one word from a sentence that was created by another player. At the end of the game, the story is revealed in its entirety.



Cat Adoption



This was a final project for BIT2008: Multimedia Data Management. It simulates a mock cat adoption site where users can browse cats and apply to "adopt".


See the code on Github

How it works

This application was made using Django. User/cat data is stored in an SQLite database unique to each copy of the repository. Cats can be added to the database using the Django admin tool. Users create an account and fill out an adoption profile that they can later edit. They can browse the cats available and filter by various characteristics. Adoption requests can be made using the "adopt" button on the desired cat's profile. A user can view all of their adoption requests as well as cancel them.
